#TweetFrom
===========
Visualize from where you use Twitter, in a fun way. 

## Live
* [Demo](http://arena.palamago.com.ar/tweetFrom/) 

## Built using:
* [jQuery](http://jquery.com/) 
* [Twitter API](https://dev.twitter.com/docs) 
* [Twitter Bootstrap](http://twitter.github.com/bootstrap/) 
* [JavaScript InfoVis Toolkit](http://thejit.org/)
* ["Superhero" Bootswatch theme](http://bootswatch.com/)
